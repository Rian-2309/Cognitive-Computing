Conceptualized through the lens of the "Computational Theory of Mind", which essentially views the human minds as an information processing system.
# 1. Meaning of Cognitive Computing
Cognitive Computing is a branch of computer science that aims to simulate human thought processes, cognitive processes (such as perception, reasoning, learning, memory and decision-making) using Artificial Intelligence, Machine Learning, Natural Language Processing and Neural Networks. 

The objective is not just to make computers calculate faster, but to enable them to:
- Understand information
- Interpret meaning
- Learn from experience
- Recognize patterns
- Adapt to new situations
- Improve over time 
- Assist humans in complex decision-making

Unlike traditional systems that operate strictly based on predefined rules, cognitive systems are designed to handle ambiguity, uncertainty, and unstructured data such as text, speech, and images.
It tries to mimic how the human brains thinks, and learn.

---
# 2. Meaning of Cognition & Why it is called 'Cognitive'?
The word 'Cognitive' comes from 'Cognition'.
Cognition refers to the mental processes involved in acquiring knowledge and understanding through thought, experience, and sensory input.
It includes:
![[Pasted image 20260306233509.png|332]]
1. Perception – interpreting sensory input
2. Attention – focusing on specific information
3. Memory – storing and retrieving knowledge
4. Language – understanding and generating communication
5. Reasoning – logical thinking
6. Learning – acquiring new knowledge
7. Decision-making – selecting the best option

Cognition is what makes humans intelligent beings.
Cognitive computing attempts to replicate these processes artificially.

---
# 3. Why Do We Need Cognitive Computing?
Traditional computers are limited because:
- They require structured input
- They cannot handle ambiguity
- They do not learn unless reprogrammed
- They lack contextual understanding

However, real-world problems are:
- Complex
- Uncertain
- Context-dependent
- Data-heavy

Therefore, we need systems that can:
- Process unstructured data
- Learn continuously
- Provide intelligent assistance

# 4. Evolution of Cognitive Computing
The **evolution of cognitive computing** refers to the development of computer systems from simple machines that perform calculations to intelligent systems that can simulate human thinking.

Main stages:
## A. Mechanical Computing Era (1600–1940)
- Early machines performed only basic calculations using **mechanical parts like gears, levers and wheels. 
- They had no intelligence or learning ability.

- **Pascaline** – mechanical calculator by Blaise Pascal
- **Difference Engine** – automatic calculation machine

**Characteristics**
- Manual operation
- Mechanical components
- Limited calculations

# 5. Difference Between Traditional and Cognitive Computing

| Traditional Computing   | Cognitive Computing       |
| ----------------------- | ------------------------- |
| Rule-based              | Learning-based            |
| Structured data only    | Structured + Unstructured |
| Deterministic           | Probabilistic             |
| No adaptation           | Self-adaptive             |
| Programmed intelligence | Simulated intelligence    |

---
# 6. Difference: Artificial Intelligence vs Cognitive Computing

| Artificial Intelligence   | Cognitive Computing          |
| ------------------------- | ---------------------------- |
| Goal: Replace human       | Goal: Assist human           |
| Automated decision making | Human-like reasoning support |
| Works independently       | Works with humans            |
| Task-focused              | Context-aware                |

# 7. Characteristics of Cognitive Systems
## Learning Ability
- A cognitive system must learn from historical data and improve over time.
## Context Awareness
- It must understand meaning or situation based on context, not just keywords.
## Reasoning Capability
- It should analyze multiple possibilities and derive logical conclusions.
## Natural Interaction
- It should communicate using natural language.
## Handling Unstructured Data
- It must process text, speech, images, and videos.

# 8. Components of Cognitive Computing

Cognitive systems are built using several technologies:
## Machine Learning (ML)
Enables systems to learn from data.
### Types:
- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning
## Natural Language Processing (NLP)
Allows machines to understand human language.
Includes:
- Text analysis
- Speech recognition
- Sentiment analysis
## Neural Networks
- Inspired by the human brain.  
- Used in deep learning systems.
## Deep Learning
- Advanced form of Machine Learning using layered Neural Networks.
## Pattern Recognition
Identifies patterns in:
- Images
- Speech
- Data
## Data Mining
Extracts useful information from large datasets.

# 9. How Cognitive Computing Works
Step-by-step process:
1. Input (Text, Voice, Image, Data)
2. Data Processing
3. Pattern Analysis
4. Reasoning and Learning
5. Decision Suggestion
6. Feedback and Improvement

Example:
- A patient reports symptoms →  
- System analyzes symptoms →  
- Compares with medical data →  
- Suggests possible diagnosis.

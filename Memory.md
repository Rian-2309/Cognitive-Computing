*Memory* refers to the ability of retaining information and reproducing it over a period of time when required to perform a cognitive task. 
>Memory = Storing and Recalling information

# Stages of Memory
It has been conceptualized as a process comprised of three stages: 
Encoding → Storage → Retrieval
All information received by our senses go through these stages.

## 1️. Encoding
Encoding is the process of **converting sensory input into a format that can be stored in memory (brain)**.
Example:
Reading a book and understanding its meaning.

## 2️. Storage
Storage refers to **maintaining encoded information in memory for a certain period of time**.
Example:
stored memory for a certain period of time in memory.

## 3️. Retrieval
Retrieval is the process of **accessing stored information when it is needed**.
Example:
retrieve answer in Exam.

However, any issue or hindrance in the completion of any of these stages can lead to memory failure.

# Types of Memory
Memory can generally divided into **3 main types**.
## 1. Sensory Memory
- This is the initial stage of memory where information from the environment is briefly stored.
- It has a very short duration, lasting only a few seconds or milliseconds.
- Sensory memory acts as a buffer, allowing us to perceive the world as a continuous stream of information.
### Characteristics
| Feature  | Description                 |
| -------- | --------------------------- |
| Duration | Few milliseconds to seconds |
| Capacity | Very large                  |
| Type     | Raw sensory data            |

## 3. Long-Term Memory (LTM)

![https://ars.els-cdn.com/content/image/3-s2.0-B9780128038130000076-f07-09-9780128038130.jpg](https://ars.els-cdn.com/content/image/3-s2.0-B9780128038130000076-f07-09-9780128038130.jpg)

- Long-term memory stores information for long periods, potentially for a lifetime.
- It has a virtually unlimited capacity.
### A. Explicit Memory (Declarative Memory)
**Explicit memory** refers to memories that we **consciously remember and can explain in words**.
> It involves **intentional recall of information**.

Two main types:
#### 1. Episodic Memory
- Memory of **personal experiences or events**.
- Includes **time, place, and context**.

Examples:
- Your **last birthday celebration**
- A **trip with friends**
- Your **first day at college**

Key idea:
> Episodic memory = remembering **events from your life**.

Brain area strongly involved:
- Hippocampus
    
#### 2. Semantic Memory
- Memory of **facts, knowledge, and general information**.
- Not linked to personal experiences.

Examples:
- **Paris is the capital of France**
- **2 + 2 = 4**
- Knowing what a **computer** is

Key idea:
> Semantic memory = **general knowledge about the world**.
### B. Implicit Memory (Non-Declarative Memory)
**Implicit memory** refers to memories that **influence behavior without conscious awareness**.
> You use the memory but **do not consciously think about it**.

Two main types here:
#### 1. Procedural Memory
- Memory of **skills and actions**.
- Learned through **practice and repetition**.

Examples:
- Riding a **bicycle**
- **Driving a car**
- **Typing on a keyboard**
- Playing **guitar**

Key idea:
> Procedural memory = **how to do things (skills)**.

Important brain area:
- Basal Ganglia
#### 2. Priming
Priming occurs when **previous exposure to a stimulus influences how we respond later**.

Examples:
- If you read the word **“doctor”**, you recognize **“nurse”** faster.
- Seeing a **brand logo** makes you choose that brand later.

Key idea:
> Priming = **past experience unconsciously influences perception or response**.

| Semantic    | Facts and knowledge                | Capital of India                 |
| Procedural  | Skills and actions                 | Riding a bike                    |

# Models of Memory in Cognitive Psychology
Memory models explain **how information is stored, processed, and retrieved in the human brain**. 
Several important models describe the structure and functioning of memory.
## 1. Multi-Store Model (Atkinson–Shiffrin Model)
- Proposed by **Richard Atkinson** and **Richard Shiffrin** in 1968, known as 'Stage model of memory' or 'modal model'.
- This model is greatly influenced by the working of computer.
- Memory is divided into three separate stores.
### Structure
| Memory Store            | Function                          | Duration            |
| ----------------------- | --------------------------------- | ------------------- |
| Sensory Memory          | Holds sensory information briefly | milliseconds        |
| Short-Term Memory (STM) | Temporary storage                 | about 20–30 seconds |
| Long-Term Memory (LTM)  | Permanent storage                 | very long time      |
### Process
```text
Stimulus → Sensory Memory → Short-Term Memory → Long-Term Memory
```

Example:  
You see a phone number → remember it briefly → store it in long-term memory.

## 2. Working Memory Model
Proposed by **Alan Baddeley** and **Graham Hitch** (1974).
### Idea
Short-term memory is **not a single system**; it has multiple components that actively process information.

The **Working Memory Model** explains how the brain **temporarily stores and processes information while performing tasks** like learning, reasoning, and comprehension.
> Working memory is a **temporary system that holds and manipulates information for short periods**.

Example:
- Solving a math problem
- Reading and understanding a sentence
- Remembering a phone number briefly
### Components of the Working Memory Model
The model has **four main components**.
#### A. Central Executive
The **Central Executive** is the **control system** of working memory.
##### Functions
- Directs attention
- Coordinates other components
- Decides which information to process

Example:  
When solving a math problem, the central executive decides:
- what information to focus on
- which process to perform next

Key idea:
> Central Executive = **attention controller of working memory**

#### B. Phonological Loop
The **Phonological Loop** processes **verbal and auditory information**.
##### Parts
1. **Phonological Store** – holds spoken words temporarily
2. **Articulatory Rehearsal Process** – repeats information to keep it in memory

Example:
- Remembering a **phone number**
- Repeating a **word in your mind**

Key idea:
> Phonological Loop = **speech and sound memory**

#### C. Visuospatial Sketchpad
The **Visuospatial Sketchpad** processes **visual and spatial information**.
##### Functions
- Stores images
- Handles spatial relationships
- Helps with mental visualization

Example:
- Remembering a **map**
- Visualizing the **layout of a room**
- Solving **puzzles**

Key idea:
> Visuospatial Sketchpad = **visual and spatial memory**

#### D. Episodic Buffer
The **Episodic Buffer** was later added by **Alan Baddeley** in **2000**.
##### Functions
- Integrates information from different sources
- Connects working memory with long-term memory

Example:  
When reading a story, it combines:
- words
- images
- past knowledge

Key idea:
> Episodic Buffer = **integration system linking working and long-term memory**

### Structure of the Working Memory Model

```
                 Central Executive
                        │
        ┌───────────────┼───────────────┐
        │               │               │
Phonological Loop   Visuospatial    Episodic Buffer
 (verbal info)       Sketchpad      (integration)
                    (visual info)
```

## 3. Levels of Processing Model
The **Levels of Processing Model** explains **how the depth of processing information affects memory retention**.
It was proposed by:
- **Fergus Craik**
- **Robert Lockhart** in **1972**.
### Idea
> The deeper we process information, the better we remember it.

This model refutes the claim of Atkinson and Shiffrin model that, memory consists of different subsystem. According to the model of level of processing (LOP), whether information will be retrieved successfully or not depends on its level of processing. LOP refers to the level at which information have been encoded.
### Three Levels of Processing
Craik and Tulving (1975) have proposed three levels of processing information:
#### A. Shallow Processing (Physical/Structural Level)
- This is the **most basic level** of processing.
- Encoding of information based on its physical attributes.
##### Characteristics
- Focuses on **physical features** of information.
- Processing is **superficial**.
- Memory retention is **weak**.
##### Examples
- Noticing whether a word is written in **capital or lowercase letters**.
- Looking at the **font style or color**.

Example question:
- Is the word **“HOUSE”** written in uppercase?

#### B. Intermediate Processing (Phonemic Level)
- This level focuses on **sound patterns** of words.
- It encoded based on how it sounds.
##### Characteristics
- Processes how a word **sounds**.
- Memory retention is **moderate**.
##### Examples
- Determining if words **rhyme**.
- Repeating a word aloud.

Example question:
- Does the word **“cat” rhyme with “hat”**?

#### C. Deep Processing (Semantic Level)
- This is the **deepest and most effective level** of processing.
- Encoding done based on its memory and/or context.
##### Characteristics
- Focuses on the **meaning of information**.
- Involves **understanding and associations**.
- Produces **strong and long-lasting memories**.
##### Examples
- Understanding the meaning of a sentence
- Connecting new knowledge with existing knowledge

Example question:
- Is a **dog a type of animal**?

Studies on this model have suggested that deeper the level of processing, the higher will be its probability to be retrieved successfully.
### Structure of the Levels of Processing
```text
Shallow Processing
(appearance of information)

↓
Phonemic Processing
(sound of information)

↓
Deep Processing
(meaning of information)
```

### Advantages of the LOP Model
- Explains **why meaningful learning improves memory**.
- Emphasizes **understanding instead of rote memorization**.
- Useful in **education and learning strategies**.
### Limitations of the LOP Model
- Hard to measure **depth of processing objectively**.
- Does not clearly explain **short-term vs long-term memory structure**.

## 4. Parallel Distributed Processing (PDP) Model
The **Parallel Distributed Processing Model** explains memory as a **network of interconnected units (neurons)** that process information **simultaneously**.
This model is also called the **Connectionist Model**.
It was mainly developed by:
- **David Rumelhart**
- **James McClelland**
### Main Idea of PDP Model
> Memory is stored in **patterns of connections between neurons**, not in a single location.

Information is processed **in parallel across many neural connections**. So memory works like a **network**, similar to **artificial neural networks in AI**.
### Key Features of PDP Model
#### A. Parallel Processing
Many pieces of information are **processed at the same time**.
Example:  
When you see a dog, the brain processes:
- shape
- color
- movement
- sound
all **simultaneously**.

#### B. Distributed Representation
Information is stored **across many neurons**, not in one place.
Example:  
The concept **“dog”** is represented by multiple features like:
- four legs
- fur
- barking
- animal
These features are stored **in different parts of the network**.
#### C. Connection Strength (Weights)
Connections between neurons have **different strengths**.
- Strong connections → information remembered easily
- Weak connections → information harder to recall
Learning happens when **connections become stronger**.
#### D. Pattern Recognition
The brain recognizes objects by **matching patterns of activation**.
Example:  
Even if you see a **different type of dog**, your brain still recognizes it as a dog because the **pattern of features is similar**.
### Structure of the PDP Model
```text
Input Units (receive information)
    ↓
Hidden Units (process the information)
    ↓
Output Units (produce the final response)
```
### Example
Suppose you see the word **“CAT.”**
1. Visual input enters the system
2. Different neurons process features like **letters, shape, sound**
3. The network activates related concepts
4. Brain recognizes it as **“cat”**
### Advantages of PDP Model
- Explains **learning through experience**
- Similar to **biological neural networks**
- Useful for **AI and machine learning**
### Limitations of PDP Model
- Difficult to explain **exact memory locations**
- Complex to model fully

## 5. Tulving’s Memory Model (Types of Long-Term Memory)
The **Tulving Memory Model** explains that **Long-Term Memory (LTM)** is not a single system, but is divided into **different types of memory based on their function**.
It was proposed by **Endel Tulving**.
### Main Idea
> Long-term memory consists of **multiple systems**, each responsible for different kinds of information.

### Structure of Tulving’s Model
```text
Long-Term Memory
│
├── Episodic Memory (events)
├── Semantic Memory (facts)
└── Procedural Memory (skills)
```

# Memory in Cognitive Computing
**Cognitive computing** systems (like AI that mimics human thinking) use memory concepts inspired by **human memory** to **store, learn, and recall information**.
> **Memory in cognitive computing = how an AI system stores knowledge, learns from data, and uses past experience to make decisions.**

## Types of Memory in Cognitive Computing
These are inspired by human memory systems:
### 1. Short-Term Memory (Working Memory)
- Stores **temporary information**
- Used during **active processing**
#### Example:
- Chatbots remembering your **last message**
- AI solving a **current problem step-by-step**
Similar to the **Working Memory Model** (by Alan Baddeley)
### 2. Long-Term Memory
- Stores information for a **long time**
- Used for **knowledge and experience**
#### Example:
- AI storing **trained data**
- Knowledge bases like **facts, rules, patterns**
### 3. Episodic Memory (Experience-Based)
- Stores **past events or interactions**
#### Example:
- A system remembering:
    - previous conversations
    - past user behavior
Similar to human episodic memory (linked to Hippocampus)
### 4. Semantic Memory (Knowledge-Based)
- Stores **facts, meanings, concepts**
#### Example:
- AI knowing:
    - “Paris is capital of France”
    - Definitions of words
Used in:
- Knowledge graphs
- Ontologies
### 5. Procedural Memory (Skills)
- Stores **how to perform tasks**
#### Example:
- AI knowing:
    - how to classify images
    - how to recommend products
## How Memory Works in Cognitive Systems
Cognitive systems follow a process:
```text
Input → Processing → Storage → Retrieval → Decision
```
1. **Input** → data from user/environment
2. **Processing** → analyze using algorithms
3. **Storage** → save in memory (database/model)
4. **Retrieval** → access stored information
5. **Decision** → generate output or action
## Technologies Used for Memory
### 1. Neural Networks
- Store information as **weights and connections**
- Inspired by brain neurons
- Used in **deep learning**
### 2. Knowledge Representation
- Stores structured knowledge using:
    - rules
    - graphs
    - ontologies
Used in:
- Expert systems
- Semantic web
### 3. Databases & Knowledge Bases
- Store large amounts of information
- Used for **retrieval and reasoning**
### 4. Machine Learning Models
- Learn from past data
- Improve performance over time

# Difference Between Memory in AI & Cognitive computing

> **AI memory = technical data storage & learning**  
> **Cognitive computing memory = human-like memory simulation**

## 1. Memory in AI 🤖
Focus: **Efficiency and learning from data**
### Characteristics
- Stores data in:
    - models (weights in neural networks)
    - databases
- Learns patterns using algorithms
- Not necessarily human-like
### Example
- Image recognition system
- Recommendation systems

Goal: **Solve tasks efficiently**
## 2. Memory in Cognitive Computing 🧠💻
Focus: **Mimicking human thinking and memory**
### Characteristics
- Inspired by human memory types:
    - Episodic (experiences)
    - Semantic (knowledge)
    - Procedural (skills)
- Tries to simulate:
    - reasoning
    - context understanding
    - human-like decisions
### Example
- Virtual assistants
- Advanced chat systems

Goal: **Think and behave like humans**

## Main Differences

| Feature     | AI Memory                  | Cognitive Computing Memory     |
| ----------- | -------------------------- | ------------------------------ |
| Approach    | Data-driven                | Human-inspired                 |
| Structure   | Neural networks, databases | Human memory models            |
| Focus       | Accuracy & performance     | Understanding & reasoning      |
| Memory Type | Not always categorized     | Episodic, semantic, procedural |
| Goal        | Solve problems             | Simulate human cognition       |
## Relationship Between Them
- Cognitive computing is a **subset/advanced form of AI**
- It uses AI techniques but adds **human-like memory concepts**

```text
Artificial Intelligence
        ↓
Cognitive Computing
        ↓
Human-like Memory Models
```

## 5. Simple Example
### Chatbot
- **AI memory:** stores trained data and generates answers
- **Cognitive memory:** tries to remember:
    - past conversations (episodic)
    - meanings (semantic)
    - how to respond (procedural)

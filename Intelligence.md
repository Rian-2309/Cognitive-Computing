# Definition of Intelligence
Intelligence is the **ability to learn from experience, adapt to new situations, understand complex ideas, solve problems and use knowledge effectively**.
>**Intelligence = Learning + Problem Solving + Adaptation**
# Theories of Intelligence
## 1. Spearman’s Two-Factor Theory
Charles Spearman proposed:
- **g-factor (general intelligence)**
- **s-factor (specific abilities)**
## 2. Thurstone’s Primary Mental Abilities
Thurstone rejected g-factor and proposed **7 abilities**:
- verbal comprehension
- numerical ability
- memory
- reasoning
- perceptual speed
- word fluency
- spatial ability
Thurstone tells intelligence is not single, it is combination of multiple abilities.

# Intelligence Quotient (IQ)
IQ is a measure of a person's intellectual abilities compared to others.
## IQ Formula
>IQ = (Mental Age / Chronological Age) × 100
## Types of Tests
| Test                | Description                  |
| ------------------- | ---------------------------- |
| Stanford-Binet Test | Individual intelligence test |
| WAIS                | Adult intelligence test      |
| Group tests         | Multiple people test         |
## IQ Classification
| IQ Range | Level                   |
| -------- | ----------------------- |
| 130+     | Very superior           |
| 110–129  | Above average           |
| 90–109   | Average                 |
| 70–89    | Below average           |
| <70      | Intellectual disability |
